# gh-pages Branch

This branch is dedicated to deploying the ToReviewers website to GitHub Pages.

## Website URL
https://thu-bpm.github.io/MarkDiffusion/ToReviewers/

## Important Notice
⚠️ **DO NOT merge this branch into main**

This is an orphan branch used solely for website deployment and is completely independent from the main branch.

## Update Instructions
To update ToReviewers content:
git checkout gh-pages
# Modify files in ToReviewers/ folder
git add ToReviewers/

git commit -m "Update ToReviewers content"

git push origin gh-pages

GitHub Pages will automatically update within a few minutes.
